## 火
- ダメージを与える基本的な攻撃属性
- 可燃性のものを燃やす（木、草、オイル）
- 氷を溶かす
- 金属を熱して高温状態にする
- 水で消される
- 継続ダメージ効果（燃焼）

## 水
- 火を消す
- 地面を濡らして滑りやすくする
- 雷との相互作用で感電範囲が広がる
- 氷と組み合わせて凍結効果
- オイルを薄める・流す

## 雷
- 瞬間的な高ダメージ
- 水や濡れた地面で範囲拡大
- 金属を伝導して周囲に拡散
- 敵を麻痺させる（スタン効果）
- 機械系の敵に特効

## 氷
- 敵を凍結させて動きを止める
- 水を凍らせて足場を作る
- 地面を凍らせて滑りやすくする
- 火で溶ける
- 凍結したターゲットは脆くなり物理ダメージ増加

## 風
- 軽い敵を吹き飛ばす
- 火を広げる（延焼範囲拡大）
- ガス・毒を拡散させる
- 飛び道具の軌道を変える
- 風の壁で弾を防ぐ

## 木
- 可燃性（火に弱い）
- 成長して障害物になる
- 水で成長促進
- トゲや蔦でダメージ・拘束

### 草
- 地面を覆い、移動速度を下げる
- 火をつけると延焼する
- 水で成長が早まる
- 視界を遮る

### 蔦
- 敵を拘束して動きを封じる
- 壁や天井に這わせる
- 火に弱い
- 時間経過で成長

## オイル
- 地面を滑りやすくする
- 火をつけると大爆発
- 水と混ざらない（水の上に浮く）
- 継続的な可燃性の罠として使える

## 岩
- 物理的な障害物
- 高い防御力
- 重力で落として押し潰す
- 火で加熱すると脆くなる
- 雷は通さない

## 金属
- 雷を伝導する
- 高い防御力だが重い
- 火で加熱すると触れたものにダメージ
- 磁力で操作可能（別属性として追加可能）
- 錆びる（水+時間）

## 蒸気
- 火+水で生成される
- 視界を遮る
- 高温でダメージ
- 冷えると水に戻る
- 圧力で爆発的な力を生む
- 風で拡散される

## 毒ガス
- 継続ダメージ（中毒）
- 風で拡散・移動する
- 火で燃えて無害化（または爆発）
- 水で薄まる・地面に沈む
- 視界を遮る効果
- 密閉空間で効果増大

## 酸
- 金属を溶かす（腐食）
- 岩石を侵食する
- 継続的な溶解ダメージ
- 水で薄まって弱まる
- アルカリ（特定物質）と中和反応
- 地面にダメージ床を作る

## 砂/土
- 風で飛ばされる（砂嵐）
- 水で泥になる（移動速度低下）
- 火で高温にするとガラス化
- 視界を遮る（砂嵐状態）
- 埋めて窒息ダメージ
- 岩より軽く扱いやすい

## 磁力
- 金属を引き寄せる・反発させる
- 金属製の弾丸や武器を無効化
- 雷と組み合わせて電磁力
- 金属製の敵を拘束・操作
- 金属の壁や扉を開閉
- コンパスや電子機器を狂わせる

## プラズマ
- 火+雷で生成される超高温状態
- 極めて高いダメージ
- 金属を瞬時に溶かす
- 光を発する（照明効果）
- 磁力で制御可能
- 不安定で制御が難しい

## 煙
- 火から生成される
- 視界を大きく遮る
- 風で拡散・移動する
- 窒息ダメージ（軽微）
- 火災警報を作動させる
- 水で消える

## 泡
- 水+圧力（空気）で生成
- 火を消す効果
- 地面を滑りやすくする
- 視界を遮る
- 時間経過で消える
- 軽くて風で飛ばされる

## 溶岩
- 岩+火で生成される
- 極めて高いダメージ
- 冷えると固まって岩になる
- 水で急冷されると蒸気が発生
- ゆっくり流れて地形を変える
- 可燃物を即座に燃やす

## 音波/振動
- 空気を振動させる
- クリスタル・ガラスを破壊
- 敵をスタンさせる（混乱）
- 壁を貫通して伝わる
- 水中でより強く伝わる
- 共鳴で効果増幅

## 光
- 闇を照らす
- 氷やクリスタルで反射・屈折
- レンズ効果で集光して火を起こす
- 影を生み出す
- 特定の敵（闇属性）に特効
- 植物の成長を促進

## 圧力
- 空気や水を圧縮
- 爆発的な力で吹き飛ばす
- 密閉空間で効果増大
- 真空を作り出す（吸引効果）
- 風の強化版として機能
- 物体を破砕する

## 属性相互作用まとめ

### 強化される組み合わせ
- 火 + 風 = 延焼範囲拡大
- 火 + オイル = 大爆発
- 雷 + 水 = 感電範囲拡大
- 雷 + 金属 = 伝導拡散
- 水 + 木 = 成長促進
- 風 + 毒ガス = 拡散範囲増大
- 風 + 煙 = 拡散範囲増大
- 風 + 砂 = 砂嵐生成
- 磁力 + 雷 = 電磁力強化
- 音波 + 水 = 伝達効果増幅
- 圧力 + 風 = 爆風強化
- 光 + 氷 = レンズ効果・反射

### 打ち消し合う組み合わせ
- 火 + 水 = 消火（蒸気発生）
- 火 + 氷 = 溶解
- 水 + オイル = 薄める
- 水 + 毒ガス = 無害化・沈殿
- 水 + 煙 = 消去
- 火 + 毒ガス = 燃焼・無害化
- 泡 + 火 = 消火

### 変化を起こす組み合わせ
- 水 + 氷 = 凍結
- 火 + 木 = 燃焼
- 火 + 岩 = 脆化
- 火 + 水 = 蒸気生成
- 火 + 雷 = プラズマ生成
- 火 + 岩 = 溶岩生成
- 火 + 木 = 煙生成
- 水 + 砂 = 泥生成
- 火 + 砂 = ガラス化
- 水 + 溶岩 = 蒸気爆発・岩化
- 水 + 圧力 = 泡生成
- 酸 + 金属 = 腐食・溶解
- 酸 + 岩 = 侵食

### 複雑な三属性以上の組み合わせ
- 火 + 水 + 圧力 = 蒸気爆発
- 風 + 火 + オイル = 火炎嵐
- 磁力 + 雷 + 金属 = 電磁投射
- 光 + 氷 + 火 = 集光燃焼
